# Changelog

## v1.0.3 - 2026-06-08

### 新增功能

#### 1. 新增 `useCookie` 参数

为了解决账号密码登录场景下 Cookie 拦截导致失败的问题，新增 `useCookie` 参数控制是否注入 Cookie。

**适用场景**：
- 当使用账号密码登录时，设置 `useCookie: false`，避免覆盖浏览器的登录 Cookie
- 当使用 Cookie 文件登录时，设置 `useCookie: true`（默认）

**涉及的 API**：

| API | 文件 | 说明 |
|-----|------|------|
| `viteMiddlewareProxy` | `src/proxy/vite-middleware-plugin.ts` | Vite 中间件代理插件 |
| `createVueProxyConfig` | `src/proxy/vue-proxy-config.ts` | Vue CLI 代理配置 |
| `createAutoProxyConfig` | `src/proxy/vue-proxy-config.ts` | 自动代理配置 |

**配置选项**：

```typescript
interface ViteMiddlewareProxyOptions {
  cookieFile: string;      // Cookie 文件路径（必需）
  target: string;          // 默认代理目标地址（必需）
  debug?: boolean;         // 是否启用调试模式
  useCookie?: boolean;     // 是否使用 Cookie 文件中的 Cookie 注入，默认 true
  proxyMap?: Record<string, string>;   // 自定义代理映射表
  proxyPaths?: string[];   // 需要代理的路径前缀列表
  ignorePaths?: string[];  // 需要忽略的路径列表
}
```

**useCookie 参数说明**：

| 值 | 效果 | 适用场景 |
|----|------|----------|
| `true`（默认） | 使用 Cookie 文件中的 Cookie 注入到请求中 | 使用 Cookie 文件登录 |
| `false` | 不注入 Cookie，使用浏览器发送的 Cookie | 使用账号密码登录，避免覆盖浏览器登录状态 |

**使用示例**：

**Vite 项目：**

```javascript
import { viteMiddlewareProxy } from 'dev-proxy-cookie'

viteMiddlewareProxy({
  cookieFile: './cookie.txt',
  target: 'http://10.17.53.3',
  useCookie: false,  // 禁用 Cookie 注入，使用浏览器发送的 Cookie
  debug: true,
  proxyPaths: ['/api'],
})
```

**Vue CLI 项目：**

```javascript
const { createAutoProxyConfig, createFileCookieGetter } = require('dev-proxy-cookie')

const getCookie = createFileCookieGetter('./cookie.txt')

createAutoProxyConfig({
  target: 'http://10.17.53.3:10000',
  getCookie,
  useCookie: false,  // 禁用 Cookie 注入
  debug: true,
})
```

### 迁移指南

#### 从 v1.0.2 升级到 v1.0.3

**无需修改任何代码**，useCookie 参数默认为 `true`，保持原有行为不变。

**如果需要使用账号密码登录，只需添加 `useCookie: false`**：

```javascript
// Vite 项目
viteMiddlewareProxy({
  cookieFile: './cookie.txt',
  target: 'http://10.17.53.3',
  useCookie: false,  // 新增：禁用 Cookie 注入
})

// Vue CLI 项目
createAutoProxyConfig({
  target: 'http://10.17.53.3',
  getCookie,
  useCookie: false,  // 新增：禁用 Cookie 注入
})
```
